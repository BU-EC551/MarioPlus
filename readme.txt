This is Five Heroes on Langya Mountain and we are proud to introduce our game to you.

Here is a readme file about how to compile our game.

Hardware Required: Digilent Nexys 4 FPGA Board
Software: Xilinx ISE 14.2 or Higher (14.7 Recommanded)

Hardware Connection:

1. Prepare your FPGA
2. Connect it with micro USB to your PC
3. Connect the VGA port to display device you have
4. Burn the .bit file into your board with Xilinx software and enjoy the game

Game Control

There are three scenes in the game. The control used here is basing on the five buttons on the Nexys 4 board.
1. Welcome
Left: 		left button
Right: 		right button
Start: 		middle button

2. Mario World:
Left: 		left button
Right: 		right button
Jump: 		up button
Attack: 	middle button

3. Star Wars Space
Left:		left button
Right:		right button
Up:		up button
Down:		down button
Attack:		middle button

If you have your own devices for controlling, just modify the .ucf file. For example, if you want to interface with the joystick, 
change the control signal from button to the PMOD input you use.

Game Flow:

After choosing a character, you can enjoy basic movement in Mario World.
The running Godzilla can be killed by either hitting from the head or attacking using Mario or Vader.
If you get caught, you will die and be back to the welcome scene.
Something interesting will happen if the question block is hit.
After you happen to enter the space, you must fight or you will easily get killed.
After 30 enemies are down, you will have the honor to see the boss.
If you are strong enough, you will defeat it and just move forward so the force can transfer you back to Mario World.

If you have any problems or suggestion, please contact funnytao@bu.edu. Enjoy the game!!!
